# Cloud 9 Fitness Studio Website

A modern, premium, and fully responsive website for **Cloud 9 Fitness Studio**, built with React and Vite.

## Prerequisites

Before you begin, ensure you have the following installed on your Mac or PC:
- **Node.js** (version 16 or higher recommended)
- **npm** (Node Package Manager)

## How to Run the Website Locally

Follow these straightforward steps to start the application on your computer:

1. **Open your Terminal** (You can find it using Spotlight by pressing Cmd + Space and typing "Terminal").
2. **Navigate to the project directory** by running:
   ```bash
   cd "/Users/apple/Documents/GYM website"
   ```
3. **Install the required dependencies** (only needed the first time, or if packages change). Run:
   ```bash
   npm install
   ```
4. **Start the development server** by running:
   ```bash
   npm run dev
   ```
5. **View the website in your browser**:
   Once the server has started successfully, it will display a local URL in the terminal (usually `http://localhost:5173/`).
   Hold down the `Cmd` key and click the link, or copy and paste that URL into your web browser (Chrome, Safari, etc.) to view the live website.

---

## Additional Commands

- **Build for Production**: 
  ```bash
  npm run build
  ```
  This will create an optimized, minified production build inside a new `dist` folder, ready to be deployed to a web host.
  
- **Preview Production Build**:
  ```bash
  npm run preview
  ```
  This allows you to preview the generated production build locally before publishing.
